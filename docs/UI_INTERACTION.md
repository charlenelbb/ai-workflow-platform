# 工作流编辑器 UI 交互说明

本文档说明优化后的界面视觉反馈与操作逻辑，便于产品与前端协作。

## 一、全局布局与视觉基调

- **配色**：主色 `#165DFF`（深蓝）、辅助色 `#36CFC9`（青蓝）、背景 `#F2F3F5`、正文 `#1D2129`。
- **圆角与阴影**：卡片/面板/节点统一 8px 圆角，`shadow-sm` 轻微阴影。
- **字体**：Inter / PingFang SC，标题 600、正文 400。

## 二、顶部导航栏

- **左侧**：产品 Logo（首字母）+ 工作流名称。点击名称可进入编辑，失焦或回车提交，Esc 取消。
- **中间**：画布操作按钮（运行 / 保存 / 导出 / 分享 / 全屏 / 对齐 / 撤销 / 重做）。运行为主色按钮，其余为描边按钮，hover 浅灰背景。
- **右侧**：用户头像，点击可展开下拉（设置 / 退出）。
- **反馈**：保存中 / 运行中时对应按钮显示「保存中…」「运行中…」并禁用。

## 三、左侧工作流列表面板

- **可折叠**：点击边缘折叠按钮可收起面板，仅保留窄条与展开图标；再次点击展开。
- **标题**：「工作流」16px 粗体。
- **新建工作流**：主色填充、6px 圆角，点击创建新工作流并自动选中。
- **列表项**：
  - Hover：背景 `#E8F3FF`。
  - 选中：背景 `#E6F7FF`，文字 `#165DFF`。
  - 右侧「⋮」点击展开下拉：重命名、删除。
- **重命名**：列表项进入编辑态，输入框获得焦点；回车提交，Esc 或失焦（空则取消）退出编辑。

## 四、中间画布区域

- **背景**：浅灰点阵（dots），点色 `#86909C`，营造网格感。
- **添加节点栏**：画布上方一排按钮（开始 / 节点 / 输入 / AI / 输出 / HTTP / 条件 / 多分支 / 结束），点击在画布上添加对应节点。
- **节点**：
  - **开始**：主色 `#165DFF` 背景，播放图标，文字「开始」在图标下方，12px 圆角、16px 内边距。
  - **结束**：危险色 `#F53F3F` 背景，停止图标，文字「结束」，样式同开始。
  - **AI 节点**：紫色 `#722ED1`；**数据节点（输入/输出）**：绿色 `#00B42A`；其他节点按类型区分颜色，统一 8px 圆角、12px 内边距，图标 + 名称。
  - Hover：轻微上浮（`translateY(-2px)`）+ 阴影增强。
  - 选中：蓝色描边高亮（2px `#165DFF`）。
- **连线**：默认 `#86909C`、2px；选中时高亮 `#165DFF`；终点带箭头指示方向。
- **交互**：点击画布空白取消节点选中；点击节点打开右侧配置面板（若有）。

## 五、右侧运行面板

- **可拖拽宽度**：左侧边缘拖拽可调整宽度，默认 320px，范围约 280–560px。
- **标题**：「运行面板」16px 粗体；右上角**运行状态**：
  - 未运行：灰色
  - 运行中：蓝色
  - 成功：绿色
  - 失败：红色
- **运行历史**：有历史时展示列表，点击某次运行可查看该次日志/Outputs/Raw。
- **标签页**：日志 / Outputs / Raw，下划线样式，选中项主色下划线。
- **日志区**：空状态提示「暂无日志。点击「运行当前工作流」开始。」；有日志时按节点卡片展示，成功/失败用不同颜色区分。

## 六、底部运行输入区

- **默认收起**：仅显示「运行输入 (JSON)」标题与展开箭头。
- **展开**：点击标题栏展开，显示 JSON 输入框与「运行当前工作流」主色按钮；再次点击标题栏收起。
- **运行**：填写合法 JSON 后点击按钮，触发当前工作流运行，结果在右侧运行面板展示；运行中按钮显示「运行中…」并禁用。

## 七、响应式与可访问性

- 顶部操作栏在小屏下部分按钮仅保留图标（文字 `hidden sm:inline`）。
- 画布添加节点栏同样在小屏下部分文字隐藏。
- 面板折叠后可为画布腾出更多空间，适配不同分辨率。

## 八、操作流程简述

1. **新建/选择工作流**：左侧「新建工作流」或点击列表项。
2. **编辑画布**：中间添加节点、拖拽连线、点击节点配置。
3. **保存**：顶部「保存」或依赖自动保存（若已实现）。
4. **运行**：底部展开运行输入 → 填写 JSON → 运行当前工作流；或顶部「运行」使用当前输入。
5. **查看结果**：右侧运行面板查看日志 / Outputs / Raw 及运行状态与历史。
